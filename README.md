# 🌱 Agro360

A plataforma inteligente para a agricultura portuguesa.

## Missão

Ajudar agricultores, cooperativas e consultores a gerir explorações agrícolas de forma simples, inteligente e baseada em dados.

## Objetivos

- Gestão de explorações
- Gestão pecuária
- Simulador de apoios
- Projetos agrícolas
- Inteligência Artificial
- Gestão financeira
- Alertas inteligentes
